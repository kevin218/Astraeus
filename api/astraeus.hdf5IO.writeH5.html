writeH5¶

astraeus.hdf5IO.writeH5(filename, verbose=True, **kwargs)¶

Save keyword arguments to a generic HDF5 file.

Parameters¶

filename: str

File name to save data, with our without extension

verbose: boolean

Set True to enable print statements declaring success/failure, and optional error message

**kwargs: number, array

Parameters to save

Returns¶

success: boolean

Return True is file was saved successfully
